# Творцы РФ — Личный кабинет участника

Рабочая реализация личного кабинета фестиваля «Творцы РФ 2026» по дизайн-хендоффу
из Claude Design (проект «Творцы (ЛК)»).

## Источники дизайна

Объединены два дизайна из одного бандла:

- **Структура и флоу** — «Творцы ЛК — Итерация 3» (`lk3/it3-*.jsx`):
  регистрация (шаг 1: email + пароль) → подтверждение email → онбординг
  (шаг 2: данные участника, валидация даты рождения 14–35) → стена 14–17 для
  несовершеннолетних → кабинет (лимит 2 заявки, черновики, отзыв с подтверждением) →
  форма заявки (01 Номинация и работа → 02 Материалы → 03 Команда → 04 Согласия,
  анкета не в форме — участник-бар) → успех подачи. Бизнес-правила — из
  handoff-листа Итерации 3.
- **Стили** — «Творцы ЛК — Основные экраны» (`lk3/fig-*`, `lk3/screens-*`):
  язык лендинга TV.fig — белый фон без «стакана», поля 16px, гаттеры 12px,
  mono-кикеры (Fira Mono), Golos Text крупно с минус-трекингом, JetBrains Mono
  для меты, sky-кнопки r16, плиты r36, тёмные полотна-питчи #15151A,
  пиксельные мозаики.

Правки заказчика из чатов учтены: «Национальность» (не «Гражданство»),
приглашения в команду бессрочные (без дат), материалов может быть несколько,
несовершеннолетний сам загружает согласие в своём кабинете, тонкие линии (1px).

## Запуск

```bash
npm install
npm run dev      # дев-сервер
npm run build    # прод-сборка в dist/
```

## Демо-логика

Всё работает на клиенте (localStorage). Кнопка **demo** в правом нижнем углу —
сценарии (новый пользователь, пустой кабинет, кабинет с заявками, командная
заявка, несовершеннолетний) и действия модератора (принять документы,
продвинуть статус, вернуть на доработку, оборвать загрузку файла).

Вход в демо-аккаунт: `m.sokolova@mail.ru` + любой пароль.
Код подтверждения email — любые 4 цифры.
