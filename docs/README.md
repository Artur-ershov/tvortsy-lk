# Юридические документы — Творцы РФ 2026

Три HTML-файла с текстами для сайта и ЛК. Вёрстка минималистичная: белый фон,
Golos Text, токены из `src/styles/tokens.css`.

## Как смотреть

Открой файл напрямую в браузере — работают без сервера.

На dev-сервере доступно по адресам:
- `/docs/polozheniye.html`
- `/docs/privacy.html`
- `/docs/consent.html`

(Vite отдаёт `public/docs/` как статику; исходники в `docs/` — зеркало,
при правке нужно скопировать в `public/docs/`.)

## Файлы

| Файл | Содержание | Где используется в ЛК |
|------|------------|----------------------|
| `polozheniye.html` | Положение о фестивале (правила участия, номинации, сроки) | Ссылка в подвале формы заявки |
| `privacy.html` | Политика конфиденциальности | Ссылка при регистрации |
| `consent.html` | Согласие на обработку персональных данных (для несовершеннолетних — подпись законного представителя) | Экран `minor-wall`, скачать/подписать |
